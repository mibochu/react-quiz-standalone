## 🚀 로컬에서 실행하기

\`\`\`bash
# 1. 저장소 클론
git clone https://github.com/yourusername/react-quiz-standalone.git
cd react-quiz-standalone

# 2. 의존성 설치
npm install

# 3. 개발 서버 실행
npm run dev
\`\`\`
